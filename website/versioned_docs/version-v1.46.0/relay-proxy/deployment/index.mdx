## Overview

GO Feature Flag relay-proxy is a standalone service that acts as a proxy between your application and the GO Feature Flag configuration.

The relay-proxy is a lightweight service that can be deployed in various environments, including Kubernetes, AWS Lambda, and more.
This page provides an explanation of several methods for deploying the relay proxy.
